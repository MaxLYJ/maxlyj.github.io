# Featured Recommendations Pages Template

Use this directory to add pages for the Featured & Recommendation section.

## Required folder naming
- Folder format: `fr-<page-slug>/`
- Example: `fr-cyber-city/`

## Required file naming inside each page folder
- `fr_<page-slug>__main.<ext>`
- `fr_<page-slug>__thumb_01.<ext>`
- `fr_<page-slug>__thumb_02.<ext>`
- `fr_<page-slug>__thumb_03.<ext>`
- `fr_<page-slug>__thumb_04.<ext>`
- Optional metadata: `fr_<page-slug>__meta.json`

Allowed image extensions: `jpg`, `jpeg`, `png`, `webp`, `svg`

## Starter template
A starter page exists at:
- `fr-template-page/`

Copy that folder and rename both folder and file prefixes to your target `<page-slug>`.
